Vol.03 — パッケージ / 1898

缶詰が
アートになった日

1898 Campbell's Soup Can / Herberton Williams

赤と白。筆記体のロゴ。金のメダル。このデザインは1898年からほぼ変わっていない。なぜ125年間、誰も手を加えなかったのか。

Work
Campbell's Condensed Soup Can Label, 1898 Herberton Williams / Joseph Campbell Preserve Co.

もともとキャンベルのスープ缶は青とオレンジのラベルだった。それを1898年、会社の幹部Herberton Williamsがコーネル大学のフットボールユニフォームの赤と白を見て「これだ」と直感し、カラーを変更した。

筆記体の「Campbell's」ロゴ、中央の金色のメダル（1900年のパリ万博で受賞した本物）、下部の「SOUP」の文字。このレイアウトはその後125年間、基本構造を変えていない。スーパーマーケットの棚で瞬時に「キャンベル」とわかる視覚的アイデンティティが、そこに宿っていたからだ。

そして1962年、アンディ・ウォーホルがこの缶を32点のシルクスクリーンで描いた。ウォーホルは「毎日同じものを食べ続けた」と語っていた。変わらないデザインが、アートの素材になった。パッケージデザインが美術館に収蔵された瞬間だった。

Warhol との関係

ウォーホルの「Campbell's Soup Cans」（1962年）は現在MoMAに収蔵されている。ウォーホルは「量産されたものの美しさ」を描いたが、その対象として選んだのは、まさに「変わらないデザイン」だった。オリジナルのパッケージデザインが変わらなかったからこそ、アートとして成立した。

"I used to drink it. I used to have the same lunch every day, for twenty years."

— Andy Warhol（キャンベルのトマトスープについて）

Today's Question

「変えない」という決断は、デザインの敗北か、それとも最大の自信の表れか？
